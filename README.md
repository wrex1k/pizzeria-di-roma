## Overview

**Pizzeria di Roma** is a full-stack web application built using **Spring Boot MVC**, **MySQL**, and **Tailwind CSS**.

The system simulates a real pizza restaurant where customers can browse the menu, create orders and manage their accounts, while administrators manage products, users and incoming orders.


### How to Run the Project

-   **Clone repo:** `git clone https://github.com/wrex1k/pizzeria-di-roma`
-   **Setup database:** execute `db_full.sql` inside the mysql interface
-   **Run application:** run class `PizzeriaDiRomaApplication` or setup maven with `mvn spring-boot:run`
